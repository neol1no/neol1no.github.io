Riot API Lookup

League of Legends Stats & Match History

Overview

A web application that utilizes the Riot Games API to fetch and display detailed player statistics for League of Legends. Users can search for summoners to view their rank, champion mastery, and recent match history with in-depth game analysis.

Key Features

  • Player Search

    Lookup any summoner by name and tag.

  • Detailed Stats

    View rank, win rates, and KDA.

  • Match History

    Recent games with champion info and outcomes.
