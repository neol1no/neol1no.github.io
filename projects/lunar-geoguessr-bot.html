Lunar Geoguessr Bot

Automated Tracking & Statistics

Automated Tracking

Seamlessly parses webhook messages to track user stats.

Overview

A sophisticated Discord bot that parses messages from a configured channel to track Geoguessr user statistics. It automatically saves data to a JSON file and provides slash commands for viewing leaderboards, individual user stats, and managing data backups.

Commands

/top

Show top 5 overall users with detailed breakdown.

/stats

Query stats for a specific user by name or ID.

/feature

Leaderboard for specific feature usage.

/info

View bot statistics and system health.

Key Features

  • Message Parsing

    Automatic processing of webhook messages.

  • Multi-Account

    Detects alt accounts and shared networks.

  • Auto-Backups

    Backups every 30 mins with reset detection.

  • Smart Alerts

    Notifications for suspicious activity.

Tech Stack

Node.js Discord.js JSON
