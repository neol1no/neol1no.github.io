Lunar Forum

Community Forum Application

Overview

A modern, full-stack forum application built with TypeScript, Node.js, and MongoDB. It features user authentication, thread creation, rich text editing, and real-time updates. Designed for community engagement with a clean and intuitive interface.

Key Features

  • User System

    Secure authentication and profiles.

  • Discussions

    Create threads, reply to posts, and categorize topics.

  • Modern Stack

    Scalable backend with MongoDB and Node.js.
