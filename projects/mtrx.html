MTRX WARE

CS2 External Cheat Base

Open Source

Overview

MTRX WARE is an external cheat base for Counter-Strike 2, written in C#. It leverages the ImGui library for a lightweight, responsive overlay. The project focuses on modularity, making it easy to add new features like aim assistance.

The software reads game memory externally to gather entity data without injecting code into the game process, minimizing detection risk. It utilizes transparent overlays to draw visual elements (ESP) directly on top of the game window.

  • Box, Skeleton, Line, Name, Distance ESP
  • Customizable Colors (Team, Enemy, Bone, Name)
  • Performance Mode & Auto Offset Updater
