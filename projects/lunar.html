Lunar Geoguessr

Advanced utility suite for Geoguessr players

Overview

Lunar is a comprehensive userscript designed to enhance the Geoguessr experience. By intercepting API calls, it enables certain features that give you an edge above others.

Key Features

  • Location: Display Country, State, City, Map.
  • Integration: Open Google Maps, PlonkIT.
  • Automation: Auto Play, Guess Mode (Region/Country/Random).
  • Misc: Hotkey system, Themes, Menu Color Customization.
